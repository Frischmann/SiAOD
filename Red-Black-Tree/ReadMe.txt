Запуск:
Требования к ПО: Иметь Python версии 3.x.x

1) Разархировать файлы в одну папку
2) Запустить файл startRBT.py
3) Ввести любое количество чисел, через пробел, для добавления их на дерево, нажать Enter
4) Выбрать любое действие с деревом:
	4.1) Добавить элемент: введите любое число для последующего добавления его на дерево
	4.2) Удалить элемент: введите любое число для последующего удаления его с дерева
	4.3) Печать дерева: R - правая ветвь, L - левая, около чисел в скобках находится его цвет: BLACK - черный, RED - красный
