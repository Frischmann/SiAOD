Запуск:
Требования к ПО: Иметь Python версии 3.x.x
1) Запустить файл startRBT.py
2) Ввести любое количество чисел, через пробел, для добавления их на дерево, нажать Enter
3) Выбрать любое действие с деревом:
	3.1) Добавить элемент: введите любое число для последующего добавления его на дерево
	3.2) Удалить элемент: введите любое число для последующего удаления его с дерева
	3.3) Печать дерева: R - правая ветвь, L - левая, около чисел в скобках находится его цвет: BLACK - черный, RED - красный
